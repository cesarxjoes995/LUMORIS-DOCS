---
title: 'Create'
api: 'POST /image-editing/nano-banana-pro-edit'
description: 'Submit an image editing job using the Nano Banana Pro -edit model.'
---

<Note>
  **Nano Banana Pro -edit** — High-quality image editing model by Google.
</Note>

### Body

<ParamField body="image" type="file" required>
  The source image to edit.
</ParamField>

<ParamField body="prompt" type="string" required>
  The text description of how to edit the image.
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.lumorislabs.online/image-editing/nano-banana-pro-edit \
  -H "Content-Type: multipart/form-data" \
  -F "image=@myimage.png" \
  -F "prompt=make it cyber-punk style"
```
</RequestExample>
