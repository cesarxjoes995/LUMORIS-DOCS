title: 'Create'
api: 'POST /api/image-editing/flux1-kontext-pro-edit'
description: 'Submit an image editing job using the FLUX.1 Kontext [pro] -edit model.'
---

<Note>
  **FLUX.1 Kontext [pro] -edit** — High-quality image editing model by FLUX.
</Note>

### Body

<ParamField body="image" type="file" required>
  The source image to edit.
</ParamField>

<ParamField body="prompt" type="string" required>
  The text description of how to edit the image.
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://disbelief-recent-thaw.ngrok-free.dev/api/image-editing/flux1-kontext-pro-edit \
  -H "Content-Type: multipart/form-data" \
  -F "image=@myimage.png" \
  -F "prompt=make it cyber-punk style"
```
</RequestExample>
