<Note>
  **Seedance 2.0 Fast** — High-quality video generation model by Seedance.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the video you want to generate.
</ParamField>

<ParamField body="aspectRatio" type="string" default="16:9">
  The aspect ratio of the generated video.
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.lumorislabs.online/video-generation/seedance-2-0-fast \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A cinematic shot of a forest", "aspectRatio": "16:9"}'
```
</RequestExample>
