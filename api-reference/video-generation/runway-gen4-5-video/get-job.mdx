---
title: 'Get task'
api: 'GET /jobs/{jobId}'
description: 'Poll the status of a Runway Gen-4.5 generation job.'
---

### Path Parameters

<ParamField path="jobId" type="string" required>
  The unique job ID.
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  The unique identifier for the job.
</ResponseField>

<ResponseField name="status" type="string">
  Current status (`queued`, `processing`, `completed`, `failed`).
</ResponseField>

<ResponseField name="videoUrl" type="string">
  The CDN URL for the generated video. Only present when `status` is `completed`.
</ResponseField>
