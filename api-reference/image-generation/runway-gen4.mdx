### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate. Be as detailed as possible to get better results.
</ParamField>

<ParamField body="aspectRatio" type="string" default="1:1">
  The aspect ratio of the image. Supported values:

  | Value  | Label      | Dimensions  |
  |--------|------------|-------------|
  | `16:9` | Widescreen | 1920 × 1080 |
  | `4:3`  | Landscape  | 1152 × 864  |
  | `1:1`  | Square     | 1080 × 1080 |
  | `3:4`  | Portrait   | 864 × 1152  |
  | `9:16` | Vertical   | 1080 × 1920 |
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the generation was successful.
</ResponseField>

<ResponseField name="imageUrl" type="string">
  The Lumoris Labs CDN URL for the generated image.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.lumorislabs.online/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A futuristic city bathed in neon lights at night",
    "aspectRatio": "16:9"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.lumorislabs.online/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: 'A futuristic city bathed in neon lights at night',
    aspectRatio: '16:9'
  })
});
const data = await response.json();
console.log(data.imageUrl);
```

```python Python
import requests

response = requests.post(
  'https://api.lumorislabs.online/generate',
  json={
    'prompt': 'A futuristic city bathed in neon lights at night',
    'aspectRatio': '16:9'
  }
)
print(response.json()['imageUrl'])
```
