---
title: 'Create'
api: 'POST /image-generation/flux-kontext-max'
description: 'Submit an image generation job using the FLUX Kontext Max model.'
---

<Note>
  **FLUX Kontext Max** — Highly contextual image generation with maximum details.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate.
</ParamField>

<ParamField body="aspectRatio" type="string" default="16:9">
  The aspect ratio of the generated image. This determines the dimensions of the final output. Supported values include:
  - `16:9` (Widescreen)
  - `4:3` (Landscape)
  - `1:1` (Square)
  - `3:4` (Portrait)
  - `9:16` (Vertical)
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.lumorislabs.online/image-generation/flux-kontext-max \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A futuristic datacenter", "aspectRatio": "16:9"}'
```
</RequestExample>
