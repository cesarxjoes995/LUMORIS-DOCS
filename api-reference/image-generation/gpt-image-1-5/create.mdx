---
title: 'Create'
api: 'POST /api/image-generation/gpt-image-1-5'
description: 'Submit an image generation job using the GPT Image 1.5 model.'
---

<Note>
  **GPT Image 1.5** — High-quality image model developed by Open Ai.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate.
</ParamField>

<ParamField body="aspectRatio" type="string" default="1:1">
  The aspect ratio of the generated image. This determines the dimensions of the final output. Supported values include:
  - `3:2` (Wide)
  - `1:1` (Square)
  - `2:3` (Tall)
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/gpt-image-1-5 \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A stunning digital art piece", "aspectRatio": "16:9"}'
```
</RequestExample>
