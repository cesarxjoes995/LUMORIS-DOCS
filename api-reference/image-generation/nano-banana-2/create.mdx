---
title: 'Nano Banana 2'
api: 'POST /api/image-generation/nano-banana-2'
description: 'Submit an image generation job using the Nano Banana 2 model.'
---

<Note>
  **Nano Banana 2** — High-quality model developed by Google.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate.
</ParamField>

<ParamField body="aspectRatio" type="string" default="16:9">
  The aspect ratio of the generated image. This determines the dimensions of the final output. Supported values include:
  - `8:1` (Cinematic banner)
  - `4:1` (Panorama)
  - `21:9` (Ultra Wide)
  - `16:9` (Widescreen)
  - `5:4` (Classic)
  - `4:3` (Landscape)
  - `3:2` (Wide)
  - `1:1` (Square)
  - `4:5` (Standard)
  - `3:4` (Portrait)
  - `2:3` (Tall)
  - `9:16` (Vertical)
  - `1:4` (Vertical banner)
  - `1:8` (Vertical strip)
</ParamField>

<ParamField body="quality" type="string" default="1k">
  The base grid quality. Options:
  - `1k` (Base grid of 1024x1024)
  - `2k` (Base grid of 2048x2048)
  - `4k` (Base grid of 4096x4096)
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/nano-banana-2 \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A futuristic google datacenter", "aspectRatio": "16:9"}'
```
</RequestExample>
