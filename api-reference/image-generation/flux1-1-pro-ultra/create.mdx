---
title: 'FLUX1.1 Pro Ultra'
api: 'POST /api/image-generation/flux1-1-pro-ultra'
description: 'Submit an image generation job using the FLUX1.1 [pro] Ultra model.'
---

<Note>
  **FLUX1.1 [pro] Ultra** — High-quality base model for stunning cinematic and detailed generations.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate.
</ParamField>

<ParamField body="aspectRatio" type="string" default="16:9">
  The aspect ratio of the generated image. This determines the dimensions of the final output. Supported values include:
  - `16:9` (Widescreen)
  - `4:3` (Landscape)
  - `1:1` (Square)
  - `3:4` (Portrait)
  - `9:16` (Vertical)
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/flux1-1-pro-ultra \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A futuristic datacenter", "aspectRatio": "16:9"}'
```
</RequestExample>
