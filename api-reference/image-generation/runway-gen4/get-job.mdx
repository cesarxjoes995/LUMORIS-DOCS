---
title: 'Get task'
api: 'GET /jobs/{jobId}'
description: 'Poll the status of a generation job. When status is `completed`, the imageUrl will be present.'
---

### Path Parameters

<ParamField path="jobId" type="string" required>
  The unique job ID returned from the Create Job endpoint.
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  The unique identifier for the job.
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the job. One of: `queued`, `processing`, `completed`, `failed`.

  | Status | Description |
  |--------|-------------|
  | `queued` | Job is waiting to be picked up |
  | `processing` | Generation is actively running |
  | `completed` | Image is ready — `imageUrl` is present |
  | `failed` | Generation failed — `error` is present |
</ResponseField>

<ResponseField name="model" type="string">
  The model used for this job (e.g. `runway-gen4-image`).
</ResponseField>

<ResponseField name="aspectRatio" type="string">
  The aspect ratio used for generation.
</ResponseField>

<ResponseField name="imageUrl" type="string">
  The Lumoris Labs CDN URL for the generated image. Only present when `status` is `completed`.
</ResponseField>

<ResponseField name="error" type="string">
  Error message if generation failed. Only present when `status` is `failed`.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the job was created.
</ResponseField>

<ResponseField name="completedAt" type="string">
  ISO 8601 timestamp of when the job finished (completed or failed).
</ResponseField>

```bash cURL
curl https://api.lumorislabs.online/jobs/c2ce03d4-bb66-4b55-bf24-8f7d046a2896
```

```javascript JavaScript
// Poll every 3 seconds until complete
async function pollJob(jobId) {
  while (true) {
    const res = await fetch(`https://api.lumorislabs.online/jobs/${jobId}`);
    const job = await res.json();

    if (job.status === 'completed') return job.imageUrl;
    if (job.status === 'failed') throw new Error(job.error);

    await new Promise(r => setTimeout(r, 3000));
  }
}
```

```python Python
import requests, time

def poll_job(job_id):
    while True:
        res = requests.get(f'https://api.lumorislabs.online/jobs/{job_id}')
        job = res.json()
        if job['status'] == 'completed':
            return job['imageUrl']
        if job['status'] == 'failed':
            raise Exception(job['error'])
        time.sleep(3)
```
