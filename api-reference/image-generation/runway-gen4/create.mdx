### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate. Be as detailed as possible for best results.
</ParamField>

<ParamField body="aspectRatio" type="string" default="1:1">
  The aspect ratio of the generated image.

  | Value  | Label      | Dimensions  |
  |--------|------------|-------------|
  | `16:9` | Widescreen | 1920 × 1080 |
  | `4:3`  | Landscape  | 1152 × 864  |
  | `1:1`  | Square     | 1080 × 1080 |
  | `3:4`  | Portrait   | 864 × 1152  |
  | `9:16` | Vertical   | 1080 × 1920 |
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/runway-gen4 \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A futuristic city bathed in neon lights at night",
    "aspectRatio": "16:9"
  }'
```

```javascript JavaScript
const response = await fetch('https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/runway-gen4', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: 'A futuristic city bathed in neon lights at night',
    aspectRatio: '16:9'
  })
});
const { jobId } = await response.json();
// Now poll /api/jobs/${jobId} every 2-3 seconds
```

```python Python
import requests

res = requests.post(
  'https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/runway-gen4',
  json={'prompt': 'A futuristic city at night', 'aspectRatio': '16:9'}
)
job = res.json()
print(job['jobId'])   # use this to poll
```
