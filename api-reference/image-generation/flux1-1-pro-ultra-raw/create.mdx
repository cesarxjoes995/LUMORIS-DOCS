title: 'Create'
api: 'POST /api/image-generation/flux1-1-pro-ultra-raw'
description: 'Submit an image generation job using the FLUX1.1 [pro] Ultra Raw model.'
---

<Note>
  **FLUX1.1 [pro] Ultra Raw** — State-of-the-art flagship model by Black Forest Labs.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The detailed text description of the image you want to generate. The FLUX1.1 [pro] Ultra Raw model excels at highly complex, descriptive prompts. Be as specific as possible about the subject, environment, lighting, and camera angles to get the best results.
</ParamField>

<ParamField body="aspectRatio" type="string" default="1:1">
  The aspect ratio of the generated image. This determines the dimensions of the final output. Supported values include:
  - `16:9` (Widescreen)
  - `4:3` (Landscape)
  - `1:1` (Square)
  - `3:4` (Portrait)
  - `9:16` (Vertical)
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://disbelief-recent-thaw.ngrok-free.dev/api/image-generation/flux1-1-pro-ultra-raw \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A hyper-realistic photo", "aspectRatio": "16:9"}'
```
</RequestExample>
