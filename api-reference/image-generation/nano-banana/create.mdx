---
title: 'Nano Banana'
api: 'POST /image-generation/nano-banana'
description: 'Submit an image generation job using the Nano Banana model.'
---

<Note>
  **Nano Banana** — High-quality model developed by Google.
</Note>

### Body

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate.
</ParamField>

<ParamField body="aspectRatio" type="string" default="1:1">
  The aspect ratio of the generated image. This determines the dimensions of the final output. Supported values include:
  - `21:9` (Ultra Wide)
  - `16:9` (Widescreen)
  - `5:4` (Classic)
  - `4:3` (Landscape)
  - `3:2` (Wide)
  - `1:1` (Square)
  - `4:5` (Standard)
  - `3:4` (Portrait)
  - `2:3` (Tall)
  - `9:16` (Vertical)
</ParamField>

### Response

<ResponseField name="jobId" type="string">
  Unique identifier for the generation job. Use this to poll for results.
</ResponseField>

<ResponseField name="status" type="string">
  Initial job status. Always `queued` on creation.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.lumorislabs.online/image-generation/nano-banana \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A futuristic google datacenter", "aspectRatio": "16:9"}'
```
</RequestExample>
